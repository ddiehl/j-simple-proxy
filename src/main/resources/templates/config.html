Usage help
Origin should be the URI that you want to be proxied, eg: /test/new.
Destination is the Full URL where you want the call to be proxied, eg: http://www.google.com/

You can use wildcards(*) in the origin url. For example if you want everything that starts with /test to be proxied you can use /test/*.
You can also use named Parameters {parameter_name} to pass content from the origin to destiny. Example: Origin- /test/{name}, destination: http://www.google.com/q={name}.
